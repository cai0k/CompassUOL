# 🚀 Compass UOL - Data Engineering Journey

## 📌 Sobre o Projeto
Este repositório contém as atividades desenvolvidas durante o programa de bolsas da Compass UOL, com foco em **Engenharia de Dados e Computação em Nuvem (AWS)**.

Ao longo do programa, foram aplicados conceitos práticos de:
- Engenharia de Dados
- ETL (Extract, Transform, Load)
- Big Data
- Cloud Computing
- Análise de Dados

---

## 🧠 Tecnologias Utilizadas
- **Python** (Pandas, NumPy)
- **SQL**
- **AWS** (S3, Lambda, Glue, Athena, EMR, Redshift)
- **Docker**
- **Apache Spark / PySpark**
- **Linux**
- **Git e GitHub**

---

## ⚙️ Principais Atividades Desenvolvidas
- Construção de pipelines ETL com Python
- Manipulação e análise de dados com Pandas e SQL
- Integração com serviços da AWS
- Processamento de dados com Spark/PySpark
- Criação de dashboards com Amazon QuickSight
- Containerização de aplicações com Docker

---

## 📊 Destaques do Projeto
- 🔹 Pipeline ETL completo (extração, transformação e carga)
- 🔹 Integração com API externa (TMDB)
- 🔹 Uso de arquitetura serverless na AWS
- 🔹 Processamento de dados em larga escala com Spark
- 🔹 Visualização de dados com QuickSight
